Marmoset, a WordPress-powered project list.

	cd path/to/wordpress/wp-content
    git clone git://github.com/abackstrom/marmoset.git
	cd plugins
	ln -s ../marmoset/marmoset-plugin .
	cd ../themes
	ln -s ../marmoset/marmoset-theme .
